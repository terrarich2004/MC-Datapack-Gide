# 📂 Создание датапаков в Minecraft: от А до Я (26.2+)

Добро пожаловать в практическое руководство по созданию датапаков для Minecraft! Этот репозиторий создан для тех, кто хочет перенести логику командных блоков в файлы, оптимизировать свои карты и освоить актуальный синтаксис версий **26.1 / 26.2**.

> [!IMPORTANT]
> Данный гайд полностью учитывает новые изменения Mojang (2026 год), включая обязательный диапазон версий (`supported_formats`) в манифесте `pack.mcmeta`.

## 🏗️ Архитектура проекта

Правильная структура папок — залог того, что игра корректно распознает ваш датапак. Внутри папки `examples/` вы найдете следующий каркас:

```text
📁 МойДатапак/
│  📄 pack.mcmeta           # Главный файл манифеста (диапазон версий)
└──📁 data/
   └──📁 minecraft/         # Пространство имен (namespace)
      ├──📁 functions/      # Скрипты с командами (.mcfunction)
      ├──📁 recipes/        # Кастомные рецепты крафта (.json)
      └──📁 tags/           # Автоматизация и группы (.json)
```

## 📋 Базовые возможности

### 1. Манифест `pack.mcmeta` (Обновлено для 26.2)
В современных версиях используется гибкий диапазон поддерживаемых форматов:
```json
{
  "pack": {
    "description": "Базовый датапак",
    "pack_format": 81,
    "supported_formats": {
      "min_inclusive": 80,
      "max_inclusive": 85
    }
  }
}
```

### 2. Функции (`.mcfunction`)
Заменяют цепочки командных блоков. Команды пишутся построчно и **без** косой черты `/`.
* `minecraft:main` — выполняется каждый тик (20 раз в секунду).
* `minecraft:init` — срабатывает один раз при загрузке мира или `/reload`.

### 3. Рецепты и теги (`.json`)
* **Recipes:** Позволяют создавать кастомные крафты на верстаке (пример крафта динамита из палок и камня лежит в папке `recipes`).
* **Tags:** Объединяют группы блоков или сущностей. Позволяют одной командой взаимодействовать, например, со всеми видами стрел или досок сразу.

## 🛠️ Инструменты разработчика

Для комфортного написания кода датапаков настоятельно рекомендуется использовать:
* **💻 Редактор:** [Visual Studio Code](https://microsoft.com)
* **🧩 Расширение:** `Data-pack Helper Plus` (подсвечивает синтаксис Minecraft, автодополняет команды и проверяет ошибки на лету).
